# Group Tutorial Roadmap

- 尽快独立完成基础科研计算任务

## Tutorial Template

每篇教程建议统一按这个模板写，方便组内持续积累：

### 标题
- 教程名称

### 1. 适合对象
- [ ] 完全新手
- [ ] 有基础但没做过这个方向
- [ ] 已做过基础计算，想系统补全

### 2. 前置知识
- [ ] 需要提前掌握的命令/软件/理论

### 3. 学完后能做什么
- [ ] 明确写出 2-4 个可落地目标

### 4. 教程正文建议结构
- [ ] 背景：这个问题是干什么的
- [ ] 核心概念：最少必要原理
- [ ] 环境准备：软件、依赖、输入文件
- [ ] 最小可运行示例
- [ ] 实战案例
- [ ] 常见报错与排查
- [ ] 结果如何判断是否正确

### 5. 配套材料
- [ ] 示例输入文件
- [ ] 示例脚本
- [ ] 示例输出结果
- [ ] 推荐参考资料

### 6. 常见坑
- [ ] 参数设置错误
- [ ] 环境变量/依赖问题
- [ ] 路径与文件格式问题
- [ ] 收敛失败或结果异常

### 7. 产出物
- [ ] 博客文章
- [ ] 配套代码仓库或脚本目录
- [ ] FAQ 条目
