# 1.15 终端环境优化

目标：在已经开始高频使用命令行之后，逐步把自己的环境整理顺。

## 1. 最基础的是 alias

```bash
alias ll='ls -lh'     # 给 ls -lh 起一个简短别名
alias gs='git status' # 给 git status 起一个简短别名
```

这类优化的意义不是“好看”，而是减少输入量和出错率。

## 2. `.zshrc`

如果你用的是 `zsh`，很多配置都写在 `~/.zshrc` 里。

你可以把它理解成：终端启动时会读取的一份个人配置文件。

## 3. `oh-my-zsh` 和插件

常见组合包括：

- `oh-my-zsh`
- `zsh-autosuggestions`
- `zsh-syntax-highlighting`
- `fzf`

这些工具主要是在帮你：

- 减少输入
- 提示历史命令
- 高亮命令语法
- 更快找文件和历史记录

## 4. 我的建议

先掌握基本命令，再做环境优化。不要一开始把时间都花在美化终端上。

## 5. 这一节最少要做到什么

1. 知道 `alias` 在做什么
2. 知道 `~/.zshrc` 是配置入口
3. 知道 `oh-my-zsh`、`fzf` 这些工具存在
