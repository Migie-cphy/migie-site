# 1.17 软链接

目标：先知道软链接能解决什么问题，不需要一开始就讲很深。

## 1. 什么是软链接

软链接很像 Windows 里的快捷方式。

```bash
ln -s /home/user/file.txt /home/user/links/mylink # 创建软链接
```

## 2. 它适合什么场景

- 给很长的路径做一个短入口
- 给常用目录建一个顺手的跳转点
- 不复制文件本体，只建立引用

## 3. 这一节最少要做到什么

1. 知道 `ln -s` 是创建软链接
2. 知道软链接不是复制文件本体
3. 知道它适合做路径入口和快捷引用
