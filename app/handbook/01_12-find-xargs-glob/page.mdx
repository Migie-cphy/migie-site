# 1.12 批量查找与处理文件

目标：目录一复杂时，不要再手工一个一个找文件。

## 1. `find`

```bash
find . -name "*.out"         # 查找当前目录下所有 out 文件
find . -type d -name "calc*" # 查找名字以 calc 开头的目录
```

当目录层级变深之后，`find` 会非常有用。

## 2. 通配符

```bash
ls *.vasp             # 列出当前目录下所有 vasp 文件
cp */CONTCAR summary/ # 复制各子目录中的 CONTCAR 到 summary 目录
```

这类写法的重点不是记所有细节，而是先理解：目录结构一旦统一，很多重复操作都能一次做掉。

## 3. `xargs`

```bash
find . -name "*.out" | xargs grep error # 查找所有 out 文件并搜索 error
```

可以先把它理解成：把前一个命令找到的一批结果，继续交给下一个命令处理。

## 4. 这一节最少要做到什么

1. 会用 `find` 找文件
2. 知道通配符可以批量匹配文件
3. 看得懂 `find ... | xargs ...` 这种基本结构
