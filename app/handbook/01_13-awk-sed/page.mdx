# 1.13 文本处理：awk 和 sed

目标：先知道这两个工具在干什么，不用第一天就学得很深。

## 1. `awk`

```bash
awk '{print $1}' file     # 输出文件的第一列
awk -F: '{print $1}' file # 以冒号为分隔符输出第一列
```

可以先把 `awk` 理解成“很适合按列取数据”的工具。

## 2. `sed`

```bash
sed 's/old/new/g' file # 将文件中的 old 替换成 new
```

可以先把 `sed` 理解成“很适合批量替换文本”的工具。

## 3. 为什么要先知道它们

很多结果提取脚本、参数替换脚本，底层其实就是 `awk` 和 `sed` 在工作。

你不一定马上手写复杂命令，但最好尽早知道：

- 提列可以想到 `awk`
- 批量替换可以想到 `sed`

## 4. 这一节最少要做到什么

1. 看得懂最简单的 `awk` 取列
2. 看得懂最简单的 `sed` 替换
3. 知道以后遇到提取和替换问题，可以往这两个方向查
