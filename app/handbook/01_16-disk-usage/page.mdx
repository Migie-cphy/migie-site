# 1.16 磁盘空间查看

目标：文件越来越多之后，知道怎么快速判断空间问题。

## 1. 最常用的两个命令

```bash
du -sh . # 查看当前目录总大小
du -sh * # 查看各子目录大小
df -h    # 查看磁盘空间使用情况
```

## 2. 它们分别在看什么

- `du`：看目录占了多大
- `df`：看整个磁盘还剩多少空间

## 3. 什么时候会用到

常见场景包括：

- 结果目录越来越大，不知道是谁占满了空间
- 程序报磁盘写入失败，怀疑盘快满了
- 想先判断哪些目录最值得清理或打包

## 4. 这一节最少要做到什么

1. 会用 `du -sh *` 看各子目录大小
2. 会用 `df -h` 看磁盘整体空间
3. 知道空间问题要先查，不要靠猜
