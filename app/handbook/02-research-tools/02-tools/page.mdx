# 2.2 常用工具

这一节不想把工具列成一张很长的清单，而是先建立一个基本感觉：科研里除了主程序本身，还会不断碰到一批配套工具。它们不一定每个都复杂，但会非常高频地出现在日常工作里。早点知道这些东西的存在，后面会少走很多弯路。

前面 Linux 和终端那一章，更像是在解决“你能不能开始动手”的问题；这一章则更像是在回答“开始做事之后，手边还会不断碰到哪些工具”。比如画图、写作、远程工具、编辑器、下载和整理资料，这些事情单独看都不算主线，但加起来会占掉你大量时间。

## 计划内容

- 编辑器与 IDE
- 远程连接工具
- 下载与传输工具
- 终端增强和效率工具
- 科研协作与文档工具
