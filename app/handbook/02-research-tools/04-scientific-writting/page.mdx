# 2.4 科研写作与 LaTeX

写论文时，真正麻烦的通常不是“会不会写一句英文”，而是整套工作流是否顺手：公式怎么排，图片怎么插，参考文献怎么管，多人怎么协作修改，最后怎么把稿件稳定编译出来。LaTeX 在这些问题上很常见，也很值得尽早建立基本直觉。

## 适合对象

- 第一次接触论文排版的同学
- 已经在写文章，但流程还比较乱的同学

## 计划内容

- LaTeX 的基本结构
- 公式、图表和交叉引用
- BibTeX / BibLaTeX 文献管理
- Overleaf、本地编辑器和 Git 协作
