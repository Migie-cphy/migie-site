# 2.2 科研里的 Python：先会这几点就够了

这一节主要讲科研里真正会用到的 Python 基础：读写文件、批量改输入、提取结果、做简单分析和画图。目标不是把 Python 讲成一门完整语言课，而是先让你知道它在组内日常计算里大概是怎么出现的。

## 适合对象

- 第一次系统接触 Python 的同学
- 已经会一点语法，但还不会把它用到科研里的同学

## 计划内容

- 文件读写与路径处理
- 批量遍历目录和改参数
- 从输出文件中提取结果
- 表格数据处理
- 结合 `matplotlib` 做最基本的科研绘图
