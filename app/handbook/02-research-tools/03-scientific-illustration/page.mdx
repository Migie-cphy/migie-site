# 2.3 科研绘图与 matplotlib

科研绘图不是“把图画出来”这么简单。更实际的问题通常是：图要表达什么、横纵坐标怎么选、不同数据怎么放在一张图里、最后怎么输出到论文里。`matplotlib` 并不是唯一工具，但它几乎是 Python 科研绘图的默认起点。

## 适合对象

- 已经会一点 Python，准备开始画图的同学
- 需要把结果整理成论文图、汇报图的同学

## 计划内容

- 折线图、散点图和柱状图
- 多面板图的组织方式
- 字体、线宽、图例和导出格式
- 怎样把图从“能看”变成“能交”
