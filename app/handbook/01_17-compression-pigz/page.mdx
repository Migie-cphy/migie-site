# 1.17 压缩、解压和并行压缩

目标：在已经会用 `tar` 的基础上，知道 `pigz` 是什么、什么时候值得用。

## 1. 基础压缩与解压

```bash
tar -czvf results.tar.gz results/ # 压缩 results 目录
tar -xzvf results.tar.gz          # 解压 results.tar.gz
```

## 2. 什么是 `pigz`

`pigz` 可以简单理解成“并行版的 gzip”。

当目录很大、文件很多时，它通常会比普通 `gzip` 更快。

```bash
tar -cf - results | pigz > results.tar.gz # 使用 pigz 并行压缩目录
pigz -d -c results.tar.gz | tar -xf -     # 解压 pigz 压缩结果
```

## 3. 不用一开始就追求这一层

如果你现在还是新手，先把 `tar` 用熟更重要。

`pigz` 更像是“知道有这个工具，等目录大了再用”。

## 4. 这一节最少要做到什么

1. 知道 `pigz` 是并行压缩工具
2. 知道它和 `tar` 经常配合使用
3. 不会把它和最基础的压缩命令混在一起
