# 1.11 最常见的压缩与解压：tar

目标：先掌握最常见的打包与解压命令，够用就行。

## 1. 最小用法

```bash
tar -czvf dir.tar.gz dir # 将目录压缩成 tar.gz
tar -xzvf dir.tar.gz     # 解压 tar.gz 文件
```

如果你现在完全是新手，先把这两条记住就够了。

## 2. 什么时候会用到

高频场景包括：

- 结果目录太多，需要打包备份
- 需要把一整个目录传给别人
- 下载到一个压缩包，需要先解压再看内容

## 3. 这一节最少要做到什么

1. 会压缩一个目录
2. 会解压一个 `tar.gz`
3. 看到 `tar.gz` 不再陌生
